# Bestiary

Because every system has beasts that need taming - shell profiles that bite, git configs that refuse to cooperate, and aliases that disappear when you need them most.

A Witcher once said:
>"Lesser, greater, middling... if I'm to choose between one evil and another, then I prefer not to choose at all."

But I’ve already chosen the lesser of two evils, and here’s the result.


## Quick Start

```bash
git clone https://github.com/Ciabas/bestiary.git ~/dotfiles
cd ~/dotfiles
./install.sh
```

### Install Packages

Add to `brew/Brewfile` and run `brew bundle` or `./brew/install.sh`.
